In signal processing, the bilinear transform creates a digital design from an analog prototype. Bilinear Audio creates original software instruments and effects, inspired by the most iconic designs of the past.

</FeatureRow>

We believe in using the best available technology to build products that are high-quality and run efficiently. We proudly use Rust, TypeScript, and the [Conformal framework](https://www.russellmcc.com/conformal/). For more on the specific techniques that went into these products, see [the Conformal App Notes](https://www.russellmcc.com/conformal/app_notes/).
